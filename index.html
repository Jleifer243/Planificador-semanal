**Nota importante:** - Añade en Firebase Console -> Authentication la URL de tu proyecto GitHub Pages en “Autorizados dominios”, esto es fundamental para el correcto funcionamiento del login Google. - Siempre usa https o un servidor local en desarrollo, no abras el archivo como `file://`, porque no funciona OAUTH allí. - El botón de login asume tiene el ID `google-login-btn` y el overlay tiene el ID `loading-overlay`. - El resto del código del planner es el tuyo sin tocar.
